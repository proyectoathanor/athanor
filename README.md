El proyecto ayhanor es dashboard creativo y esteticamente útil para persivir visualmente datos de mi organización de distintos proyectos para poder tomar desiciones, automatizarlos mejor y hacer todo cada vez más productivo pero fácil de hacer.

Visualmente el proyecto athanor es como un mapa espacial dónde ciertas variables controlan el aspecto y movimiento de los cuerpos.

1-Cada proyecto es un objeto/cuerpo 
2-Cada cuerpo gira en circulo al rededor su cuerpo inicial
3-La velocidad de giro de cada cuerpo es la duración del proyecto
4-El color del cuerpo es que cuanto tiempo queda por para hacer el proyecto siendo la fecha y hora de inicio menos la fecha y hora final el 100% (color blaco), entonces un proyecto de 2 horas empezado a la 6 con fin a las 10 tiene 4-2=2horas de rango, es decir que a las 6 tiene 100%, a las 7 50% y a las 8 0%.
5-El área (espacio se expande) una unidad al día.
6-La distancia de el cuerpo y el cuerpo sobre el que orbita es la frecuencia de ejecuciónes, es decir que si es algo planeado para hacerse cada x días y no se hace entonces cada día que no se haga se aleja empezando en 1unidad x=x+1, el primer día 1unidad, el segundo 3unidades, el tercer día 6 unidades. 
7-El tamaño esta determinado por la importancia contextual, es decir una unión de todos los objetivoses son en contexto, y que tanto ayuda o perjudica ese proyecto al contexto es lo que determina el tamaño
